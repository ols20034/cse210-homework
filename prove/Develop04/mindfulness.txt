What does the program do?
    program that provides activites to provide different types of Mindfulness
    opportunities through prompts, guidence, and journaling


How does program work?
    -> user prompted to enter number for which activity they want to do
    <- user enters a number from 1-3


Breathing activity:
    -> program prompts user for duration of activity
    <- user enters number for whole seconds of duration
    -> program begins with beginning message (breath in)
        each prompt counts 5 seconds
    -> promgram prompts (breath out)
        prompt counts down 5 seconds
    -> program loops for duration user entered
    -> end of time - displays motivating message " Heavenly Father beleives in you"

Reflection activity
    -> program prompts random message and prompt for activity
    -> prompt for activity reflection - 
    -> program prompts for time in seconds  
    <- user enters time for activity in seconds
    -> program displays random prompts and allows user to contimplate for 10 seconds each
    -> program ends and returns to main menu when finished

Listing activity
    -> program prompts random message and prompt for activity
    -> prompt for activity reflection - 
    -> program prompts for time in seconds  
    <- user enters time for activity in seconds
    -> program displays random prompts and allows user to contimplate for 10 seconds each
    -> program ends and returns to main menu when finished
